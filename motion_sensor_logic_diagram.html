⏱️ Времеви Настройки

1.25 сек непрекъснато движение
Вентилатор включване:
70 сек след светлината
Светлина изключване:
90 сек без движение
Вентилатор изключване:
90 сек след светлината
Boost продължitelност:
3 минути (по подразбиране)
MQTT Failsafe:
5 минути без връзка
🔄 Основна Логика на Работа
1. Инициализация
WiFi свързване, MQTT настройка, PIR калибриране, OTA активиране
2. PIR Мониторинг
Непрекъснат мониторинг на PIR сензора с debounce филтър
Debounce: 300ms
3. Движение Засечено
Стартира броячи за непрекъснато движение
4. Светлина Активация
След 1.25 сек непрекъснато движение
Delay: 1250ms
5. Вентилатор Планиране
Планира се включването на вентилатора
Delay: 70 сек
6. Автоматично Изключване
При липса на движение за определен период
Timeout: 90 сек

⚡ Специални Функции
🚀 Boost Режим
Ръчно активиране на вентилатора за определен период чрез MQTT команда
Topic: home/toilet_motion/switch/fan/boost/set
Payload: минути (1-10)
⚠️ Failsafe Режим
Локална работа без MQTT при липса на връзка над 5 минути
Timeout: 5 минути
🔧 Safety Check
Периодична проверка на състоянието на релетата и памет
Interval: 30 секунди
🔄 Reset Function
Нулиране на настройките при задържане на бутона 3 секунди
Hold: 3 секунди

🌳 Дърво на Решенията
🔍 Motion Detected?
├─ YES: Start continuous motion timer
├─ Motion > 1.25s?
├─ YES: Turn ON Light → Schedule Fan (70s)
└─ NO: Continue monitoring
└─ NO: Check timeouts
├─ No motion > 90s & Light ON?
├─ YES: Turn OFF Light
└─ Light OFF > 90s & Fan ON?
└─ YES: Turn OFF Fan
🚀 Boost Active?
├─ YES: Override normal logic, Force Fan ON
└─ NO: Follow normal sequence
⚠️ MQTT Connected?
├─ YES: Normal operation + HA integration
└─ NO: Failsafe mode (local operation only)
